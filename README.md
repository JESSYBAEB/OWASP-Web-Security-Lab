Web Application Vulnerability Assessment Using OWASP ZAP

Overview

This project was completed as part of a cybersecurity internship program. The objective was to assess the security of an intentionally vulnerable web application using OWASP ZAP, identify common web application vulnerabilities, and document findings with appropriate mitigation recommendations.

Internship Task

As part of the internship, I:

* Set up a vulnerable web application (WebGoat).
* Performed vulnerability scanning using OWASP ZAP.
* Identified SQL Injection, Cross-Site Scripting (XSS), and Cross-Site Request Forgery (CSRF) vulnerabilities.
* Documented findings with supporting evidence.
* Recommended security controls to reduce identified risks.
